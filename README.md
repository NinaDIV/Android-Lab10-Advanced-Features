# Android Lab 10 - Caracteristicas Avanzadas

Laboratorio Android de nivel avanzado. Persistencia de datos con Room Database, consultas asincronas con Coroutines y arquitectura MVVM completa.

## Temas cubiertos
- Room Database (Entity, DAO, Database)
- Coroutines para operaciones asincronas
- LiveData y ViewModel
- Repositorio como capa de abstraccion de datos
- Arquitectura MVVM
