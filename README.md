# Laboratorio Android 10 — Características Avanzadas (Room, Coroutines y MVVM)

![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?style=flat&logo=kotlin&logoColor=white)
![Android](https://img.shields.io/badge/Android-3DDC84?style=flat&logo=android&logoColor=white)
![Room](https://img.shields.io/badge/Room_DB-4285F4?style=flat&logo=android&logoColor=white)

Laboratorio Android de nivel avanzado desarrollado en Kotlin. Aborda la persistencia de datos local con Room Database, las consultas asíncronas con Coroutines y la implementación completa de la arquitectura MVVM con un repositorio como capa de abstracción de datos.

## 📚 Temas cubiertos

- Room Database (Entity, DAO, Database)
- Coroutines para operaciones asíncronas
- LiveData y ViewModel
- Repositorio como capa de abstracción de datos
- Arquitectura MVVM

## 🛠️ Tecnologías

| Tecnología | Uso |
|---|---|
| Kotlin | Lenguaje principal del proyecto |
| Android SDK | Plataforma de desarrollo |
| Room Database | Persistencia de datos local (SQLite) |
| Kotlin Coroutines | Operaciones asíncronas |
| Android Jetpack (ViewModel, LiveData) | Gestión de estado y ciclo de vida |

## ✅ Requisitos previos

Antes de ejecutar el proyecto necesitas tener instalado:

- **Android Studio** (versión reciente, con el plugin de Kotlin incluido)
- **JDK 17** (requerido por las versiones actuales del Android Gradle Plugin)
- **Android SDK** (se instala desde el SDK Manager de Android Studio)
- Un **emulador Android** configurado en el AVD Manager **o** un **dispositivo físico** con la depuración USB activada

## 🚀 Instalación y ejecución

1. Clona el repositorio:

   ```bash
   git clone https://github.com/NinaDIV/Android-Lab10-Advanced-Features.git
   cd Android-Lab10-Advanced-Features
   ```

2. Abre Android Studio y selecciona **Open**, luego elige la carpeta del proyecto que quieras revisar (`Lab/`, `Tarea/` o `Distrogas/`).
3. Espera a que Gradle sincronice y descargue las dependencias automáticamente.
4. Selecciona un emulador o dispositivo físico en la barra de dispositivos.
5. Pulsa **Run** (▶) o usa `Shift + F10`. La aplicación se instalará y abrirá en el dispositivo seleccionado.
